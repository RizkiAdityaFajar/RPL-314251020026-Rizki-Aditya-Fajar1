 # Quality Attributes-Sistem Akademik Kampus
standar mutu perangkat lunak terkini dan kebutuhan khusus lingkungan perguruan tinggi, berikut atribut mutu lengkapnya:
1. Kesesuaian Fungsional
Kelengkapan: Memenuhi semua proses akademik: pendaftaran, jadwal, KRS, nilai, transkrip, kelulusan, pembayaran
Ketepatan: Data akurat, perhitungan SKS/IPK sesuai aturan, tidak ada kesalahan perhitungan
Kesesuaian: Mengikuti aturan PDDikti, Kemenag, dan standar akreditasi nasional
Kepatuhan: Sesuai regulasi perlindungan data pribadi 

2. Kinerja & Efisiensi
Waktu respons: Halaman terbuka kurang dari 2 detik, pemrosesan data cepat saat jam sibuk
Penggunaan sumber daya: Hemat memori, ruang penyimpanan, dan beban server
Kapasitas: Dapat melayani ribuan pengguna bersamaan tanpa melambat

3. Keandalan
Ketersediaan: Waktu operasi lebih dari 99%, jarang mati total; ada cadangan jika gangguan
Ketahanan kesalahan: Tetap berjalan meski ada input salah atau gangguan jaringan
Pemulihan: Bisa kembali normal & memulihkan data dalam waktu singkat jika terjadi gangguan
Konsistensi: Data tetap sama di semua perangkat/modul

4. Kegunaan 
Mudah dipelajari: Mahasiswa/dosen/staf cepat paham cara pakai
Mudah dioperasikan: Navigasi jelas, menu logis, pesan kesalahan informatif
Dapat dipahami: Istilah sesuai dunia akademik, tidak membingungkan
Aksesibilitas: Bisa dipakai lewat laptop, HP, dan browser umum

5. Keamanan
Kerahasiaan: Data hanya dibaca pihak berwenang; mahasiswa hanya lihat datanya sendiri
Integritas: Data tidak bisa diubah/hapus tanpa izin & jejak audit tercatat
Keaslian: Login aman, verifikasi identitas, hindari akses ilegal
Ketahanan serangan: Terlindung dari peretasan, virus, kebocoran data
