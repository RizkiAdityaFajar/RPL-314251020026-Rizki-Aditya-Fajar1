# Physical View-Sistem Akademik kampus

# Komponen Utama & Lokasi Fisik
- Pusat Data Kampus (Data Center)
Merupakan jantung sistem, diletakkan di ruangan khusus yang aman dan terkontrol:
Perangkat: Server utama, server cadangan, perangkat penyimpanan (NAS/SAN), sistem pendingin, UPS, genset cadangan.
Lokasi: Ruang Data Center terpisah, biasanya di gedung administrasi pusat.
Fungsi: Menyimpan seluruh basis data akademik, menjalankan aplikasi inti, mengelola akses pengguna.

- Jaringan Komunikasi
Menghubungkan semua perangkat di seluruh area kampus:
Perangkat: Router, switch, kabel LAN serat optik, titik akses WiFi, firewall, perangkat pengaman jaringan.
Jangkauan: Terhubung ke seluruh gedung kuliah, administrasi, perpustakaan dan terhubung ke jaringan internet eksternal.

- Tempat Layanan & Perangkat Pengguna
Tersebar di berbagai titik layanan:
Bagian Akademik: Komputer staf, printer jaringan, pemindai dokumen.
Laboratorium Komputer: Komputer untuk akses mahasiswa dan dosen.
Perpustakaan: Terminal pencarian data akademik dan jadwal.
Gedung Jurusan: Komputer dosen dan koordinator prodi untuk input nilai dan pengaturan jadwal.
Akses Luar Kampus: Sistem dapat diakses melalui jaringan internet dari perangkat pribadi (laptop, ponsel) mahasiswa/dosen

# Karakteristik Penting Pandangan Fisik
Keamanan: Ruang server dikunci, memiliki sistem pemantauan suhu, listrik cadangan, dan proteksi kebakaran.
Ketersediaan: Menggunakan sistem cadangan agar jika satu perangkat rusak, layanan tetap berjalan.
Konektivitas: Seluruh bagian terhubung sehingga data dapat diakses secara real-time oleh pihak yang berwenang.
